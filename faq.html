FAQ

Frequently asked questions

What is Forty-Fives?
A Maritime Canadian trick-taking card game with unusual, region-specific trump rankings.
How do I play?
Each game mode has a "How to play" panel on its page, and the Learn Ranking trainer teaches the card rankings.
In Configure for Auction, what are the rules profiles (Common PEI, Wikipedia, Rec Hall PEI, Tignish PEI)?
Each profile is a set of configuration values for a style of play. Common PEI is the default and matches how the game is most often played on PEI. Wikipedia follows the Wikipedia baseline. Rec Hall PEI is a faster kitty-less game. Tignish PEI is like Common PEI but deals a five-card kitty. You can also choose Custom and set each value yourself.
In Configure for Auction, what is "Always exchange non-trump"?
When enabled, this provides a button for convenient discard of all non-trump cards, so you don’t have to select each one.
In Configure for Auction, what is "Hide other players"?
When enabled, this removes the rendering of computer players, which is cleaner on mobile devices.
We play Auction differently than this game.
We are listening! Please let us know the differences and where you are located. See the Feedback page. We will try and incorporate the changes.
